Bootstrap 2.3

To use Backform with Bootstrap 2.3, call the bootstrap2 function. Add this at the beginging of your HTML document, after Backform is loaded: 

<script>
  Backform.bootstrap2();
</script>
This will change the global class names. In some cases, the markup must be changed. Controls can have a bootstrap2 function defined for themselves to change the markup. Look at the code for Backform.RadioControl as an example.
